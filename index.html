LOGIC FOR COMPUTERS

The most fundamental concept on digital logic is the BOOLEAN ALGEBRA. the boolean algebra is fundamental in circuit design, binary systems and programming.
- - What is Boolean Algebra? - -
Boolean algebra is a branch of mathematics that deals with variables that have two distinct values: true and false. It was introduced by George Boole in his first book, The Mathematical Analysis of Logic, in 1847. Boolean algebra forms the basis for designing and analyzing digital circuits and logic gates.

George Boole (1815 - 1864)

LOGIC OPERATORS

Logic operators are used in programming, mathematics, and logic to perform operations with logical values (true or false).
Most common logical operators:
· AND
Returns true if both operands are true.
Example: 0 AND 1 = 0
· OR
Returns true if at least one of the operands is true
Example: 0 OR 1 = 1
· NOT
Returns the inverse of the operand's value.
Example: NOT 1 = 0
· XOR
Returns true if exactly one of the operands is true.
Example: 0 XOR 1 = 1
· NAND
Returns true if at least one of the operands is false.
Example: 1 NAND 1 = 0
· NOR
Returns true if both operands are false..
Example: 1 NOR 0 = 0

LOGIC GATES

A logic gate is a device that performs a Boolean function, they are used to build digital circuits. The logic gates correspond to a specific logic operator.
